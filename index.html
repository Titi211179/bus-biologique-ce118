Découverte clé — Inhibition compétitive Amygdale ⇄ Insula

Le résultat le plus fort (rho=-0.226***) révèle que l'amygdale et l'insula sont en compétition active. Quand le stimulus visuel (faces) active fortement l'amygdale via le bypass high-gamma, l'amygdale envoie un signal inhibiteur à l'insula. Ce n'est pas un simple contournement passif — c'est une suppression active.

Cela explique pourquoi dans ta Phase 5, les faces montrent rho=-0.025 sur le bus V1→Insula : l'amygdale intercepte le signal face et bloque la voie insulaire. Les lettres (rho=0.130**) passent parce qu'elles n'activent pas l'amygdale, donc pas d'inhibition.

Cingulaire ACC (16 ch)
BURST CODING — signal prioritaire
Toutes fenêtres. Le cingulaire antérieur (hub du réseau de saillance avec l'insula) utilise le burst coding — paquets de spikes rapides pour signaler la saillance.
Préfrontal (14 ch)
AMPLITUDE + SLOPE — dialecte visuel tonique
Le PFC orbitofrontal reçoit en amplitude lente. Anti-corrélé avec l'insula (rho=-0.106***).
Pariétal (16 ch)
BURST (300-800ms) / AMP+SLOPE (50-300ms)
Switch de protocole : amplitude pendant l'encodage visuel, puis burst pendant la réponse insulaire. Transition de mode de communication.

Comparaison Phase 5 (contact-par-contact) vs Bus v11.5 (région-moyenné)

Phase 5 analysait chaque contact insulaire comme cible séparée → rho=0.647 chez CE118 sur broadband.
Bus v11.5 moyenne les 44 contacts insulaires en un seul scalaire → rho=-0.056 (signal dilué).
Les deux résultats sont cohérents : le bus existe mais il est spatialement sélectif (certains contacts insulaires répondent fort, d'autres non). Pour retrouver le rho=0.647, il faudrait identifier les contacts insulaires les plus réactifs (probablement T1aIa et T1bId = insula antérieure/dorsale).
